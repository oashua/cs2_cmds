# CS2 控制台指令速查

这是一个纯前端的静态网站，用于快速检索并复制常用 CS2 控制台指令，方便个人训练。

## 功能

- 关键词搜索（指令名、指令、说明、标签）
- 按分类筛选（机器人控制、投掷物练习等）
- 收藏常用指令（本地存储）
- 一键复制单条指令、当前列表或勾选的多条指令（多条指令用 `;` 连接）
- 分享和应用直接使用原始指令文本（支持多条指令）
- 为所有指令设置快捷键，并生成 CS2 控制台可用的 `bind` 命令
- 带参数指令支持选择/输入参数，分享代码会附带参数值

## 使用方式

直接打开 `index.html` 即可使用；如需本地服务器体验，可使用任意静态服务器。

### 快捷键说明

- 点击“设置快捷键”后按任意键完成设置
- 设置时按 `Esc` 取消，按 `Backspace/Delete` 或 `` ` `` 清除已设置的快捷键
- 设置快捷键后会自动勾选，取消快捷键会自动取消勾选
- 点击“清空快捷键绑定”可一次性移除所有快捷键并清空已选

### 绑定到游戏内快捷键

点击“复制分享指令”或“复制已选/绑定命令”会生成类似以下内容：

`bind F5 "sv_cheats 1"; bind F6 "sv_grenade_trajectory 1"`

将其粘贴到 CS2 控制台即可完成绑定。

### 参数编辑

- 带参数的指令会显示输入框或选择框
- 修改参数会即时更新命令内容
- 分享代码会保留参数值

## 如何新增指令

在 `app.js` 的 `commands` 数组中新增一项：

```js
{
  name: "指令名称",
  command: "控制台指令",
  description: "用途说明",
  category: "分类",
  tags: ["标签1", "标签2"]
}
```

> 注意：部分指令需要先开启 `sv_cheats 1`。
